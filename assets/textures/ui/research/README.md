# Research Category Icons

Place the icon files for research categories in this directory.
The files should be PNG images (preferably transparent).
Recommended size: 32x32 or 64x64 pixels.

## Naming Convention

The file names must correspond to the `TechCategory` enum (snake_case):

- `electronics.png`
- `military.png`
- `space_technology.png`
- `biology.png`
- `physics.png`
- `energy.png`
- `sociology.png`
- `construction.png`
- `propulsion.png`
- `materials.png`
- `sensors.png`
- `weapons.png`
- `defensive_systems.png`
- `life_support.png`
- `industry.png`

If a file is missing, the game will fallback to displaying the category emoji.
